# Class-Only Impact Summary

- Class records now carry `classTraits` arrays and `kits[].kitTraits` arrays.
- The class page modal edits class table fields, class traits, and kit details directly.
- `Kit 0` is required on every class and stays as the default no-effect kit.
- The injector script can be rerun to rebuild class/kit traits from the current data and corpus traces.
- `data/classTraitTrace.json` is generated by the injector as a source trace report.
- Character selection and kit dropdown behavior use the normalized kit lists.
- Any future class-table imports must preserve the structured trait arrays and avoid raw JSON-only editing.
